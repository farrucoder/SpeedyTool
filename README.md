# SpeedyTool
## SpeedyTool is the free tool that provides two services. Major part of this project is the modern "Glass Morphism" UI that makes it different from others, Service first service is user can generate QR code dynamically in real time, it also downloadable, Second service is the Image Compressor without losing or compromising of image, fast and secure. 

## 🌐 Live Demo

https://speedytool.web.app

## ✨ Features

- 📸 Image Compression
- 📂 QR Code Generator
- ⚡ Fast Performance
- 🎯 User-friendly Glassmorphism UI
